使用Python引用类型的注意事项

2016/01/26

Python引用类型的使用

在Python中，存在引用类型，当把引用类型加入到集合中，需要小心，否则集合中的多个元素可能会指向内存中同一地址的内容，这样会导致当你修改集合中某一个元素时，另一个元素也随之更改，防不胜防。
例如：

tmp = ['a', 'b']
myDict = {}

myVar1 = tmp
myVar2 = tmp
myDict['key1'] = myVar1
myDict['key2'] = myVar2

myDict['key1'].extend(['c'])

print(str(myDict))

输出：
{'key2': ['a', 'b', 'c'], 'key1': ['a', 'b', 'c']}
