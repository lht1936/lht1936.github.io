    # title: "将codingiOS开源项目导入xCode并运行" date: 2016-03-16 21:22:34 tags: iOS, xCode categories: iOS

    ##安装CocoaPods CodingiOS项目使用业界标准CocoaPods管理依赖。 执行以下命令检查是否安装了CocoaPods:

    pod --version

    如果未安装，执行以下命令进行安装：

    sudo gem install cocoapods

    CocoaPods使用ruby编写，所以使用gem命令进行安装。

    ##下载CodingiOS源码 使用git下载：

    git clone git://git.coding.net/coding/Coding-iOS.git

    ##安装依赖库并运行 进入源码下载目录，执行源码自带的bootstrap命令，其包含的执行内容包括：

    #!/bin/bash
CMD_PATH=`dirname $0`
cd $CMD_PATH
cp Coding_iOS/Coding_iOS-Prefix.pch.example Coding_iOS/Coding_iOS-Prefix.pch
git submodule update --init --recursive
pod install
open Coding_iOS.xcworkspace
exit 0

    pod install即为安装依赖库，这个过程比较慢，而且需要翻墙，否则下载不下来。 需要注意的是，下载友盟库的时候，又需要切换为国内网络，因为友盟库的服务器在国内，使用vpn可能会下载不下来，看你的vpn支不支持了。
