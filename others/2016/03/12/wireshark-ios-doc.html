离线安装Xcode和iOS文档

2016/03/12

今天把Xcode从7.0.1更新到7.2.1了，想更新下离线文档，醉了天朝的网络，半天没动静，搜了下找到这篇帖子：
Download iOS SDK 7.0 documentation for offline install
用wireshark获取文档的下载地址，然后使用其他工具直接下载，更快。

Wireshark

顺便学学wireshark，官方网站很多资料，比如：
1.youtube视频
2.wireshark还会跟Apple一样搞活动，在每年6月份，名字叫SharkFest，这是他们从08年到15年的历史纪录：
* SharkFest’15
* SharkFest’14
* SharkFest’13
* SharkFest’12
* SharkFest’11
* SharkFest’10
* SharkFest’09
* SharkFest’08
3.上面都是视频类，上最通用的文档类，直接抄袭咯：
* Web pages(browseable):One huge page or mutiple pages
* Web pages(ZIP file): One huge page or multiple pages
* PDF: US or A4
4.Wireshark还有很多书籍介绍的。

使用Wireshark嗅探

1.首先在活动监视器中找到Xcode的进程id，然后在Xcode中启动下载
2.通过lsof -i | grep pid 的方式，找到Xcode所使用的下载端口号
3.在Wireshark中通过tcp.port == [port]的方式过滤出下载的tcp网络包
4.在包内容中寻找下载地址，但是发现被加密了。。

此方法失败。。

apple文档链接

最后还是使用这个文件中的地址来进行下载。。
https://developer.apple.com/library/downloads/docset-index.dvtdownloadableindex
从这个xml文件中搜索你要的版本对应的文档，比如我要下载iOS9.3的，直接搜iOS9.3，可以找到这个版本对应的iOS文档地址为：
https://devimages.apple.com.edgekey.net/docsets/20160321/031-52212-A.dmg
虽然可以直接使用这个地址下载，但速度会很慢，将edgekey.net去掉，使用apple.com站点下载即可提高速度。
小提示：请使用百度离线下载，离线下载完成后在从百度云下载，很快！！

安装dmg文件

参考以下链接,亲测有效哈：
http://stackoverflow.com/questions/29443681/how-to-install-xcode-documentation-by-dmg-file
主要是两点：
1.Xcode下载完成的目录为~/Library/Caches/com.apple.dt.Xcode/Downloads/
2.文件命名规则为”identifier string” + “-“ + “version string” + “.dmg”
例如iOS 9.3的文件名：com.apple.adc.documentation.iOS-93.5.dmg

Xcode安装或升级

如果需要安装Xcode，请去开发者中心登录下载，在app store真是太慢了！！
链接：https://developer.apple.com/downloads/
使用以下命令验证安装的Xcode来源:

spctl --assess --verbose /Applications/Xcode.app

下载Xcode 7.3，里面自带ios simulator 9.3，所以sdk不用另外再下载，除非需要其他版本。
文档还是要另外下载安装的。
