###1. for循环的范围，用..会报错，只能用…
```swift
//Operator is not a known binary operator
//Use of unresolved identifier ‘..’
var firstForLoop = 0
for i in 0..3{
firstForLoop += i
}
firstForLoop


###2. 函数调用，除了第一个参数外，其它参数需要指定参数名
```swift
func greet(name: String, day: String) -> String{
    return "hello \(name), today is \(day)."
}
//missing argument label 'day:' in call
greet("linhongting", "Saturday")

//正确调用方法
greet("linhongting", day: "Saturday")
